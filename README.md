
# Music System en BDFD v3.2

Un sistema para poder reproducir musica con tu bot de discord en Bot Desinger For Discord.


## Configuración

#### Solo necesitas crear 2 variable


| name | value     |
| :-------- | :------- |
| `TOKEN` | `El token de su bot de discord` | 
| `play` | `false` |

## Comandos
- !play
- !stop
- !vol
- $onInteraction (2)
