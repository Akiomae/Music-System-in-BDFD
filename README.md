
# Music System en BDFD

Un sistema para poder reproducir musica con tu bot de discord en Bot Desinger For Discord.


## Configuración

#### Solo necesitas crear una variable


| name | value     |
| :-------- | :------- |
| `TOKEN` | `El token de su bot de discord` | 

## Comandos
- !play
- !addMusic
- $onInteraction[play]
- $onInteraction[stop]
- $onInteraction[pause]
- $onInteraction[resume]
- $onInteraction[skip]
- $onInteraction[list]
